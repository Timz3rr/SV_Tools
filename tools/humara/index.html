HUMARA — Inactivation X

Analyse du test HUMARA (digestion HpaII + PCR) pour l'étude de l'inactivation X.
Identifie l'allèle muté, détermine l'origine parentale et évalue le statut de conductrice.

① Contexte clinique

Si coché : dans le sang d'une conductrice, l'inactivation X est biaisée vers l'allèle muté.

② Données familiales

Laissez vide si aucun garçon atteint connu dans la famille.

③ Individu à analyser

Laissez vide si les données HpaII ne sont pas disponibles.
